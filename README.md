# Time-Stopper
A proof of concept data pack and resource pack that makes use of vanilla shaders and the (at the time) new /tick command


## Instructions
Because the tick command requires elevated permissions to use, this project will only work running in a server with function-permission-level set to 3 or 4 in the server properties file.


To use, install this datapack in the world, and then run /function time:give.
To stop time, hold down the clock for approximately 1 second, and then release. To resume time, click once with the clock.

## Note:
### This project only works in Minecraft version 1.20.5. A newer release is coming soon, however due to changes in vanilla shaders, it most likely won't include the custom shader effects the 1.20.5 version has.

### In order to see the custom shader, make sure that you have graphics set to fabulous in the video settings!
